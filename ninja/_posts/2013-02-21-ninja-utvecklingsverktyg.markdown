<h1>Utvecklingsverktyg<&#47;h1></p>
<p>Innan vi b&ouml;rjar med att skapa ett wordpress-tema beh&ouml;ver vi vissa verktyg.<br&#47;><br&#47;></p>
<h2>Ett webbhotell eller server<&#47;h2><br />
F&ouml;r att Wordpress ska fungera kr&auml;vs en server eller ett webhotell med st&ouml;d f&ouml;r PHP och MySQL. I denna kurs kommer vi att g&aring; igenom hur du anv&auml;nder ett webbhotell, men du kan &auml;ven g&ouml;ra detta p&aring; en lokal server p&aring; din dator, med tex XAMPP eller MAMP. F&ouml;r att utveckla ett wordpress tema p&aring; ett webbhotell beh&ouml;ver du en tillg&aring;ng till FTP. F&ouml;r mer information om FTP, se <a href="&#47;nyborjare-publicera&#47;" target="_blank">h&auml;r.<&#47;a></p>
<p>N&auml;r du har anslutit till din sida &auml;r du redo f&ouml;r att installera Wordpress.</p>

F&ouml;r att installera Wordpress s&aring; beh&ouml;ver du ladda ner den senaste versionen och ladda upp den till ditt webbhotell. F&ouml;r mer info om hur du g&aring;r till v&auml;ga, se <a href="&#47;installera-wordpress&#47;" target="_blank">Installera WordPress.<&#47;a></p>
<h2>Exempeldata<&#47;h2><br />

N&aring;gra alternativ &auml;r:<br />

<p>Dessa filer inneh&aring;ller olika data f&ouml;r att anv&auml;nda som utfyllnad innan du har riktiga poster och dylikt p&aring; din sida. Det kan vara en bra ide att l&auml;gga till lite egen data ocks&aring;, s&aring; att du kan se hur slutresultet blir.</p>

F&ouml;r att skapa ett wordpress-tema beh&ouml;ver du ha n&aring;gorlunda kunskaper i HTML och CSS. Du kan l&auml;ra dig detta i v&aring;ra tidigare kapitel, <a href="&#47;nyborjare-inledning&#47;">Nyb&ouml;rjare<&#47;a> och <a href="&#47;kunnig-inledning&#47;">Kunnig<&#47;a>.</p>
