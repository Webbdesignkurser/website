Utvecklingsverktyg</h1>

Innan vi börjar med att skapa ett wordpress-tema behöver vi vissa verktyg.<br/><br/>

Ett webbhotell eller server</h2>
För att Wordpress ska fungera krävs en server eller ett webhotell med stöd för PHP och MySQL. I denna kurs kommer vi att gå igenom hur du använder ett webbhotell, men du kan även göra detta på en lokal server på din dator, med tex XAMPP eller MAMP. För att utveckla ett wordpress tema på ett webbhotell behöver du en tillgång till FTP. För mer information om FTP, se här.</a>

När du har anslutit till din sida är du redo för att installera Wordpress.

Wordpress</h2>
<img src="/wp-content/uploads/2013/02/ninja-wordpress-förhandvisning.png" alt="" width="695" height="227" class="alignnone size-full wp-image-3584" />
För att installera Wordpress så behöver du ladda ner den senaste versionen och ladda upp den till ditt webbhotell. För mer info om hur du går till väga, se Installera WordPress.</a>

Exempeldata</h2>
Din Wordpress installation kommer att behöva exempeldata, utfyllnad, eller liknande. I Wordpress admin-panelen, gå till Verktyg -> Importera</em> och välj Wordpress från listan av alternativ. Nu behöver vi bara en WXR fil att importera.
Några alternativ är:
The Theme Unit Test data</a>
WPCandy Sample Content</a>

Dessa filer innehåller olika data för att använda som utfyllnad innan du har riktiga poster och dylikt på din sida. Det kan vara en bra ide att lägga till lite egen data också, så att du kan se hur slutresultet blir.

En texthanterare</h2>
Du behöver inga speciella bildhanteringsprogram eller liknande för att skapa ett wordpresstema, en vanlig texthanterare räcker. Vi rekommenderar Sublime Text 2</a> vilket är delvis gratis då man måste betala för fortsatt användning. Om du hellre föredrar ett gratis program så går Notepad++</a> lika bra.</strong>

HTML och CSS</h2>
För att skapa ett wordpress-tema behöver du ha någorlunda kunskaper i HTML och CSS. Du kan lära dig detta i våra tidigare kapitel, Nybörjare</a> och Kunnig</a>.

PHP</h2>
För att skapa ett Wordpress-tema behöver du även kunna en del PHP, vilket du kommer att få lära dig i denna kursen.
