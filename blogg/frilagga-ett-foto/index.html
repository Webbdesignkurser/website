Att frilägga ett foto kan vara praktiskt om du vill byta bakgrund eller frilägga ett objekt helt enkelt. För att spara en bild som har har transparent bakgrund behöver du spara den som PNG eftersom JPG inte stödjer transparens.

2. Markera området</h2>
 </strong>Klicka på området du vill frilägga, uppe i hörnet kan du bestämma känsligheten på trollstaven.
