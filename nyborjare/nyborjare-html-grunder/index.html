HTML-grunder</h1>
HTML är själva grundstenen i din design. Här läggs all text och bild.
När man skriver i html så jobbar man med taggar < ></strong> som berättar om hur texten ska tolkas.
Det kan röra sig om länkar, texter eller bilder. Dessa html-texter sparas sedan ner som ett html-dokument via texthanteraren.<br/><br/>

Sätta upp det nödvändiga</h2>

1.</strong> Skapa en ny mapp på datorn. Här kommer vi att lägga alla filer som hemsidan ska bestå av.
2.</strong> Öppna texthanteraren och skapa ett nytt dokument/projekt. Det är här vi kommer att skriva allt innehåll.
