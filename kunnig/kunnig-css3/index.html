Vad är CSS3?</h1>
CSS3 är den senaste versionen av CSS.</strong> Det har tillkommit ett flertal nya funktioner, så som text-skugga, rundade kanter, animering, selection och mycket mer. Vi kommer att visa några av dessa genom att applicera det på vår HTML-sida. För dig som aldrig använt CSS förut så hänvisar vi till CSS-grunder</a>. Läs också HTML5-kapitlet</a> innan du går vidare.

I vår blogg</a> hittar du fler användbara funktioner som man kan åstadkomma med CSS3.<br/><br/>

1. Vi börjar med Reset CSS</h2>
Om du läst föregående kurser så vet du vad Reset gör, nämligen att nollställa alla stil-värden som webbläsaren annars skulle ha gett din hemsida. Den här processen är viktig för att hemsidan ska se korrekt ut oavsätt vilken webbläsare du använder.
Skapa ett nytt dokument och klistra in följande CSS-kod.
[css]
html, body, div, span, applet, object, iframe,
h1, h2, h3, h4, h5, h6, p, blockquote, pre,
a, abbr, acronym, address, big, cite, code,
del, dfn, em, img, ins, kbd, q, s, samp,
small, strike, strong, sub, sup, tt, var,
b, u, i, center,
dl, dt, dd, ol, ul, li,
fieldset, form, label, legend,
table, caption, tbody, tfoot, thead, tr, th, td,
article, aside, canvas, details, embed,
figure, figcaption, footer, header, hgroup,
menu, nav, output, ruby, section, sumgmary,
time, mark, audio, video {
margin: 0;
padding: 0;
border: 0;
font-size: 100%;
font: inherit;
vertical-align: baseline;
}
article, aside, details, figcaption, figure,
footer, header, hgroup, menu, nav, section {
display: block;
}
body {
line-height: 1;
}
ol, ul {
list-style: none;
}
blockquote, q {
quotes: none;
}
blockquote:before, blockquote:after,
q:before, q:after {
content: '';
content: none;
}
table {
border-collapse: collapse;
border-spacing: 0;
}
[/css]
<br/><br/>

2. Body</h2>
Skriv sedan en bakgrund för body (det som täcker hela sidan).
[css]
body{
background: #ececec;
}
[/css]

3. Wrapper</h2>
På html-sidan har vi ett flertal wrappers skrivna som classer, men vi kommer att sammanfatta dom i en och samma class, kallad .wrapper</strong>. Läs mer om Wrappers här</a>.
[css]
.wrapper{
width: 960px;
margin-left: auto;
margin-right: auto;
}
[/css]

5. Loggan</h2>
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/css3-header2.png" alt="" title="" width="700" height="88" class="alignnone size-full wp-image-2330" />
[css]
#logo{
width: 130px;
height: 29px;
background: url(logo.png);
}
[/css]
Diven #logo</strong> ska nu visa bilden på vår logga. För det här måste vi veta bredden</strong>, höjden</strong> och adressen</strong> till bilden.
För att ta reda på måtten i din bild - högerklicka på bilden > välj egenskaper > Information.
Om du har Photoshop öppet kan du även kopiera bilden och skapa nytt dockument, då ser du bildens mått i rutan.

Placera ut loggan</h2>
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/css3-header3.png" alt="" title="" width="700" height="88" class="alignnone size-full wp-image-2331" />
[css]
#logo{
width: 130px;
height: 29px;
background: url(logo.png);
position: absolute;

margin-left: 0px;
margin-top: 30px;
}
[/css]
Vi ska även placera ut #logo någonstans i header. Det här gör vi genom att lägga till position och margin-attribut i diven.

6. Menyn</h2>
Vi börjar med att placera ut #menu</strong> till höger i headern.

<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/css-meny22.png" alt="" title="" width="700" height="89" class="alignnone size-full wp-image-2449" />
[css]
#menu{
position: absolute;
margin-left: 500px;
margin-top: 35px;
overflow: hidden;
}
[/css]

Som du såg i designen tidigare så ska menyn ligga horisontellt, det här ska vi fixa med hjälp av float:left </strong>.
I html-filen har vi skrivit menyn som en lista</a> vilket tillåter oss att ändra li-taggarna</strong> så att dom lägger sig på bredden istället.

<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/css-meny31.png" alt="" title="" width="700" height="89" class="alignnone size-full wp-image-2450" />
[css]
#menu li{
float:left;
margin-left: 80px;
}
[/css]

@font-face</h2>
Nu är meny-länkarna utplacerade som vi vill, men någonting saknas, nämligen rätt typsnitt och färg</strong>.
Typsnitt kan du ladda ner lite vart som helst, det finns dom som kostar pengar</a> och dom som är gratis</a>.
När man har laddat ner en font så vill man kanske applicera den på hemsidan, det här gör man med @font-face</strong>. Det finns bra tjänster som hjälper dig att göra fonten webbanpassad, bland annat Font2web</a> och Fontsquirrel</a>.

När du laddar upp en font</strong> där så får du allt genererat åt dig, dvs. en css-kod och det typsnittet du valt i olika filformat. Alla teckensnitts-format stöds nämligen inte av alla webbläsare, man måste därför applicera en mängd olika på sidan för att vara säker på att den funkar.

<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/fontface-fonts.jpg" alt="" title="" width="700" height="172" class="alignnone size-full wp-image-2733" />
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/hagin-font.jpg" alt="" title="" width="700" height="172" class="alignnone size-full wp-image-2738" />

I det här exemplet har vi använt en gratis font som heter Hagin</a>. Här följer vår @font-face-struktur:</strong>
[css]
@font-face {
font-family: 'Hagin';
src: url('fonts/hagin_caps_medium.eot');
src: local('â˜º'),
url('fonts/hagin_caps_medium.woff') format('woff'),
url('fonts/hagin_caps_medium.ttf') format('truetype'),
url('fonts/hagin_caps_medium.svg') format('svg');
font-weight: normal;
font-style: normal;
}
[/css]

...och nu skriver vi helt enkelt in den nya fonten på våra länkar.
[css]
#menu a{
font-family: Hagin;
color:#222;
font-size:16px;
text-decoration: none;
}
[/css]

7. Headline</h2>
Headlinen har tidigare inte varit synlig i html-filen. Nu gör vi den 100% bred och 300px hög, samt färgar den ljusblå.
[css]
#headline{
background:#8fc5df;
width:100%;
height:300px;
}
[/css]
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/headline.jpg" alt="" title="" width="700" height="377" class="alignnone size-full wp-image-2735" />

Headline-bilden</h2>
Bilden ligger i en div som vi kallat #headline_bild</strong>. Genom att ändra bestämma position kan vi flytta den till höger om texten.
[css]
#headline_bild{
position: absolute;
margin-left:400px;
margin-top:35px;
}
[/css]
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/headline-position3.jpg" alt="" title="" width="700" height="245" class="alignnone size-full wp-image-2741" />

Headline-texten</h2>
Vi gör samma sak med texten. Här placerar vi den 100px</strong> ner.
[css]
#headline_text{
position: absolute;
margin-top: 100px;
}
[/css]
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/headline-position2.jpg" alt="" title="" width="700" height="245" class="alignnone size-full wp-image-2740" />

Styla texten med CSS3</h2>
Headline-texten ska bli vit med en svart skugga. Vi börjar med det grundläggande:
[css]
#headline_text h1{
font-family: Hagin;
color:#fff;
font-size: 30px;
}
[/css]
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/headline-position4.jpg" alt="" title="" width="700" height="245" class="alignnone size-full wp-image-2743" />

Nu ska vi lägga till skuggan</strong> på texten, det här gör vi med funktionen text-shadow</strong>.
Istället för webbfärger (#000) så skriver vi skuggan i rgba</strong> (red, green, blue och alpha). Vi gör det här för att kunna bestämma hur genomskinlig skuggan ska vara (alpha). Som exempel skriver jag .5</strong> på alpha, vilket ger oss en 50% genomskinlighet.
[css]
#headline_text h1{
font-family: Hagin;
color:#fff;
font-size: 30px;

text-shadow: rgba(0,0,0,.5) 0 1px 0;
}
[/css]
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/headline-position5.jpg" alt="" title="" width="700" height="245" class="alignnone size-full wp-image-2744" /><br/><br/>

8. Content</h2>
#content</strong> är uppdelad i två delar, en till vänster och en till höger.
För att lägga de två delarna bredvid varandra skriver vi overflow: hidden</strong> i #content och float: left</strong> i den vänstra och högra delen.
[css]
#content{
position: relative;
margin-top: 30px;
width: 960px;
overflow: hidden;
}

#left_box{
position: relative;
width: 700px;
float: left;
}

#right_box{
position: relative;
width: 240px;
margin-left: 20px;
float: left;
}
[/css]
<img src="http://webbdesignkurser.se/wp-content/uploads/2012/07/content-position1.jpg" alt="" title="" width="700" height="363" class="alignnone size-full wp-image-2780" />
