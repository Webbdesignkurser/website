Grunderna i Photoshop</h1>
Med photoshop kan du i princip göra vad som helst. Foto-manipulation, teckna och webbdesign förstås.
Här går vi igenom grunderna för detta fantastiska program som går att köpa eller hyra på Adobe.com</a>.<br/><br/>

Öppna programmet</h2>
Tryck Ctrl + N</strong> för att skapa ett nytt dokument. Command + N för mac.

<img class="alignnone size-full wp-image-1087" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop1.png" alt="" width="700" height="371" />

Text</h2>
Välj text-verktyget genom att antingen klicka på ikonen eller snabbkommando T</strong>.

<img class="alignnone size-full wp-image-1088" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop2.png" alt="" width="700" height="373" />

Lager</h2>
När du har skrivit din text ser du att det har skapats ett lager för din text. Du kan skapa nya lager genom att trycka på lager-ikonen. Genom att vi kan skapa lager hjälper det oss också att redigera sådant som ligger under ett annat lager. Vi kan också välja att göra ett lager osynligt genom att trycka på ögat.

<img class="alignnone size-full wp-image-1089" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop3.png" alt="" width="700" height="362" />

Lagerstilar</h2>
Om vi dubbelklickar på ett lager får vi upp lagerstilar. Här kan vi bestämma hur vårat lager ska se ut. Det finns skuggor, färg överteckning, ramar och mycket mer.

<img class="alignnone size-full wp-image-1091" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop4.png" alt="" width="700" height="400" />

Omvandla</h2>
Med kortkommandot V</strong> får du fram omvandlings-verktyget. Kryssa i rutan uppe till vänster för att ta fram kontrollen.

<img class="alignnone size-full wp-image-1092" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop5.png" alt="" width="700" height="342" />

Markera, kopiera och klistra in</h2>
Kortkommandot M</strong> tar fram markerings-verktyget. Med det här kan du markera ett område, kopiera det och klistra in någon annan stans. Du kopierar på Ctrl + C</strong> och klistrar in på Ctrl + V</strong>. För mac är det Command + C och Command + V.

<img class="alignnone size-full wp-image-1097" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop6.png" alt="" width="700" height="350" />

Lägga samman lager</h2>
Du lägger samman lager genom att högerklicka och välja ett av Lägg samman-alternativen.

<img class="alignnone size-full wp-image-1099" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop7.png" alt="" width="700" height="350" />

Lagermasken</h2>
Lagermasken består av vitt och svart, det läggs ovanpå ett redan skapat lager och används för att markera vad som ska synas av bilden. Du ritar med svart på det som inte ska synas och med vitt på det som ska synas.

<img class="alignnone size-full wp-image-1100" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop8.png" alt="" width="700" height="400" />

<img class="alignnone size-full wp-image-1101" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop9.png" alt="" width="700" height="400" />

Justeringslager</h2>
Med justeringslagret kan vi bestämma bland annat mättnad, kontrast, svärta, ljus med mera. Precis som på lagermasken kan vi måla med svart på det som inte ska synas och med vitt på det som ska synas av vårt justeringslager.

<img class="alignnone size-full wp-image-1105" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop101.png" alt="" width="700" height="565" />

Invertera lager</h2>
För att invertera en lagermask, ett justeringslager eller ett vanligt lager trycker du Ctrl + i</strong> eller Command + i om du har en mac.

<img class="alignnone size-full wp-image-1106" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop11.png" alt="" width="700" height="565" />

Spara som</h2>
Tryck Ctrl + S</strong> eller Command + S för att spara din bild. Om du sparar som PSD (photoshop dokument) sparas dina lager precis som dem är. För att publicera bilderna på nätet kan du välja mellan JPG och PNG.

<img class="alignnone size-full wp-image-1112" title="" src="http://webbdesignkurser.se/wp-content/uploads/2012/07/photoshop12.png" alt="" width="700" height="414" />
