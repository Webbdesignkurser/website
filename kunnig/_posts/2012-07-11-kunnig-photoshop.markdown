<h1>Grunderna i Photoshop<&#47;h1><br />
Med photoshop kan du i princip g&ouml;ra vad som helst. Foto-manipulation, teckna och webbdesign f&ouml;rst&aring;s.<br />
H&auml;r g&aring;r vi igenom grunderna f&ouml;r detta fantastiska program som g&aring;r att k&ouml;pa eller hyra p&aring; <a href="http:&#47;&#47;adobe.com" target="_blank">Adobe.com<&#47;a>.<br&#47;><br&#47;></p>
<h2>&Ouml;ppna programmet<&#47;h2><br />
Tryck <strong>Ctrl + N<&#47;strong> f&ouml;r att skapa ett nytt dokument. Command + N f&ouml;r mac.</p>
<p><img class="alignnone size-full wp-image-1087" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop1.png" alt="" width="700" height="371" &#47;></p>
<h2>Text<&#47;h2><br />
V&auml;lj text-verktyget genom att antingen klicka p&aring; ikonen eller snabbkommando <strong>T<&#47;strong>.</p>
<p><img class="alignnone size-full wp-image-1088" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop2.png" alt="" width="700" height="373" &#47;></p>
<h2>Lager<&#47;h2><br />
N&auml;r du har skrivit din text ser du att det har skapats ett lager f&ouml;r din text. Du kan skapa nya lager genom att trycka p&aring; lager-ikonen. Genom att vi kan skapa lager hj&auml;lper det oss ocks&aring; att redigera s&aring;dant som ligger under ett annat lager. Vi kan ocks&aring; v&auml;lja att g&ouml;ra ett lager osynligt genom att trycka p&aring; &ouml;gat.</p>
<p><img class="alignnone size-full wp-image-1089" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop3.png" alt="" width="700" height="362" &#47;></p>
<h2>Lagerstilar<&#47;h2><br />
Om vi dubbelklickar p&aring; ett lager f&aring;r vi upp lagerstilar. H&auml;r kan vi best&auml;mma hur v&aring;rat lager ska se ut. Det finns skuggor, f&auml;rg &ouml;verteckning, ramar och mycket mer.</p>
<p><img class="alignnone size-full wp-image-1091" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop4.png" alt="" width="700" height="400" &#47;></p>
<h2>Omvandla<&#47;h2><br />
Med kortkommandot <strong>V<&#47;strong> f&aring;r du fram omvandlings-verktyget. Kryssa i rutan uppe till v&auml;nster f&ouml;r att ta fram kontrollen.</p>
<p><img class="alignnone size-full wp-image-1092" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop5.png" alt="" width="700" height="342" &#47;></p>
<h2>Markera, kopiera och klistra in<&#47;h2><br />
Kortkommandot <strong>M<&#47;strong> tar fram markerings-verktyget. Med det h&auml;r kan du markera ett omr&aring;de, kopiera det och klistra in n&aring;gon annan stans. Du kopierar p&aring; <strong>Ctrl + C<&#47;strong> och klistrar in p&aring; <strong>Ctrl + V<&#47;strong>. F&ouml;r mac &auml;r det Command + C och Command + V.</p>
<p><img class="alignnone size-full wp-image-1097" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop6.png" alt="" width="700" height="350" &#47;></p>
<h2>L&auml;gga samman lager<&#47;h2><br />
Du l&auml;gger samman lager genom att h&ouml;gerklicka och v&auml;lja ett av L&auml;gg samman-alternativen.</p>
<p><img class="alignnone size-full wp-image-1099" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop7.png" alt="" width="700" height="350" &#47;></p>
<h2>Lagermasken<&#47;h2><br />
Lagermasken best&aring;r av vitt och svart, det l&auml;ggs ovanp&aring; ett redan skapat lager och anv&auml;nds f&ouml;r att markera vad som ska synas av bilden. Du ritar med svart p&aring; det som inte ska synas och med vitt p&aring; det som ska synas.</p>
<p><img class="alignnone size-full wp-image-1100" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop8.png" alt="" width="700" height="400" &#47;></p>
<p><img class="alignnone size-full wp-image-1101" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop9.png" alt="" width="700" height="400" &#47;></p>
<h2>Justeringslager<&#47;h2><br />
Med justeringslagret kan vi best&auml;mma bland annat m&auml;ttnad, kontrast, sv&auml;rta, ljus med mera. Precis som p&aring; lagermasken kan vi m&aring;la med svart p&aring; det som inte ska synas och med vitt p&aring; det som ska synas av v&aring;rt justeringslager.</p>
<p><img class="alignnone size-full wp-image-1105" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop101.png" alt="" width="700" height="565" &#47;></p>
<h2>Invertera lager<&#47;h2><br />
F&ouml;r att invertera en lagermask, ett justeringslager eller ett vanligt lager trycker du <strong>Ctrl + i<&#47;strong> eller Command + i om du har en mac.</p>
<p><img class="alignnone size-full wp-image-1106" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop11.png" alt="" width="700" height="565" &#47;></p>
<h2>Spara som<&#47;h2><br />
Tryck <strong>Ctrl + S<&#47;strong> eller Command + S f&ouml;r att spara din bild. Om du sparar som PSD (photoshop dokument) sparas dina lager precis som dem &auml;r. F&ouml;r att publicera bilderna p&aring; n&auml;tet kan du v&auml;lja mellan JPG och PNG.</p>
<p><img class="alignnone size-full wp-image-1112" title="" src="http:&#47;&#47;webbdesignkurser.se&#47;wp-content&#47;uploads&#47;2012&#47;07&#47;photoshop12.png" alt="" width="700" height="414" &#47;></p>
<h2>Anv&auml;ndbara kommandon<&#47;h2><br />
Vissa kommandon &auml;r extra anv&auml;ndbara, du kan till och med best&auml;mma kortkommandon sj&auml;lv beroende p&aring; vad som passar dig b&auml;st. Det g&ouml;r du p&aring; <strong>Redigera > Kortkommandon<&#47;strong>.</p>
